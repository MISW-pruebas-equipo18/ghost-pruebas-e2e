# Ghost Pruebas E2E

This repository stores tests for Ghost performed in Cypress and Kraken.

## How to use Cypress

Install

```bash
npm install

```

Run test

```bash
npm run cy:open

```


* Escenarios de prueba Kraken: [pruebas-kraken](./pruebas-kraken)
* Escenarios de prueba Cypress: [pruebas-cypress](./cypress)
